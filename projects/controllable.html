Fine-grained Prosody Control of TTS System (prototype web service)
Mar 2021 - Apr 2022 (@ NCSOFT)

I researched and developed the TTS system that is able to control the prosody of speech in a fine-grained level, so that the users can generate speech with the prosody they want. I contributed to the release of this fine-grained, controllable TTS system, which has been made available as an in-company web service. Notably, it was used in official guide videos for NCSOFT's games, enhancing the user experience.
